Januari dan April 2025 diproses melalui pembersihan data, pelipatan huruf besar-kecil, normalisasi, tokenisasi, penghapusan stopword,dan stemming. Pembobotan kata dilakukan menggunakan
TF-IDF, dan ulasan dikategorikan menjadi positif (peringkat >3) dan negatif (peringkat â‰¤ 3).
